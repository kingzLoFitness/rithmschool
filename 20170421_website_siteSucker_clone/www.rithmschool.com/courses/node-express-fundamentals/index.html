{ Rithm School Node and Express.js Fundamentals. }

Welcome to Rithm's free Node and Express.js fundamentals prep course! The material here will guide you through the basics of Node and Express. You'll learn how Node.js works, how to create and use built-in modules, and you'll build server-side applications using Express. You'll learn to store data in Mongo, use Mongoose for additional functionality, and deploy your applications to Heroku! By the end of the course, you'll be able to build simple Node and Express applications backed with a Mongo database.
